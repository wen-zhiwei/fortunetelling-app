# PWA图标说明

PWA需要以下图标文件：

- `/public/icon-192.png` - 192x192像素的PNG图标
- `/public/icon-512.png` - 512x512像素的PNG图标

这些图标用于：
- 主屏幕图标
- PWA安装提示
- 浏览器标签页图标

## 创建图标

你可以使用任何图像编辑工具创建像素风格的图标，建议：
- 使用像素艺术风格
- 主题色与应用的三种主题色（紫色、金色、绿色）相关
- 简洁明了的设计，适合小尺寸显示

## 临时方案

在开发阶段，你可以：
1. 创建简单的占位图标
2. 或使用在线图标生成器
3. 或暂时注释掉manifest.json中的icons配置
